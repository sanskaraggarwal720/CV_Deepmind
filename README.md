# AI Content Pipeline

This is an AI Content Pipeline mobile application built with Expo, React Native, NativeWind, and the Gemini API (NB2 Lite & Omni Flash).

## Implementation Checklist

- [x] **Phase 1:** Scaffold + Triple Spike (API, Stitch, On-Device Gemma)
- [ ] **Phase 2:** Stitch Design System + Screen 1 (Prompt & Template Select) Generation
- [ ] **Phase 3:** Screen 2 (Supervisor, Direct Build) + Gemma Decomposition Logic
- [ ] **Phase 4:** Screen 3 (Image Grid & Select) via Stitch + Image Generation (NB2 Lite)
- [ ] **Phase 5:** Screen 4 (Video Studio, Direct Build) + Video Gen/Edit (Omni Flash)
- [ ] **Phase 6:** Screen 5 (Post Composer) via Stitch + Buffer

---

*Note: Phase 1 triggers were modified based on API capabilities (Omni Flash shifted to Interactions API) and environment constraints (fallback to Gemma API instead of on-device native modules).*
